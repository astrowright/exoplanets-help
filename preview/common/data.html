Data Columns

Here is a list of all available data columns:

Column ID Name Units Description
Info Columns
KOI_SYS KOI System The system component of the KOI
KOI_OBJ KOI Object The object component of the KOI
FIRST_OBJECT First Object? If Yes, this KOI is the first object in the KOI system. Add "FIRST_OBJECT" in the Filter if you want to ensure that each KOI system appears only once in the table or plot.
N_OBJ Number of Objects The number of objects in the KOI system is a useful quantity when generating plots.
RA Right Ascension Hours Right Ascension (J2000)
DEC Declination Degrees Declination (J2000)
HAB_ZONE Habitable Zone? If Yes, this planet might be in the habitable zone. The criterion for inclusion the habitable zone is that 183 ≤ TPLANET ≤ 373. See Borucki et al for for discussion.
Planet Properties
RPLANET Planetary Radius Earth Radii Planetary Radius
A Semi-Major Axis AU Semi-major axis of orbit. This is the longer dimension of the ellipse-shaped orbit, based on Newton's generalization of Kepler's third law and the stellar mass. For a circular orbit, A is the orbit radius.
AR a/R* Ratio of semi-major axis to stellar radius. Assuming zero eccentricity, a parameter derived from the lightcurve. Note for planets in non-circular orbits, a/R* is the scaled planet-star separation at the time of transit.
UAR Uncertainty in AR Uncertainty in AR
RR r/R* Ratio of planet radius to stellar radius
URR Uncertainty in RR Uncertainty in RR
TPLANET Planet Temperature °Kelvin Equilibrium temperature of planet. See Borucki et al for discussion.
Transit Properties
T0 Time of Transit Center BJD-2454900 Time of a transit center; BJD-2454900. Based on a linear fit to all observed transits.
UT0 Uncertainty in T0 Days Uncertainty in T0
DUR Transit Duration Hours Transit duration, first contact to last contact.
PER Period Days Average interval between transits. Based on a linear fit to all observed transits.
UPER Uncertainty in PER Days Uncertainty in PER
DEPTH Transit Depth Parts-per-million Transit depth at center of transit.
IMPACT Impact Parameter Impact parameter of transit. Note that there is a strong co-variance between the Impact Parameter b and a/R*.
UIMPACT Uncertainty in IMPACT Uncertainty in IMPACT
SNR Transit Signal-to-Noise Ratio Total signal to noise ratio of all transits detected. SNR = DEPTH/(STD*sqrt(N)) where STD is the standard deviation of all data outside of transits.
Stellar Properties
RSTAR Stellar Radius Solar Radii Stellar radius reported by KIC. If TSTAR_FLAG = 1, then R is based on J-K interpolation.
MSTAR Stellar Mass Solar Masses Stellar mass derived from log(g) and stellar radius.
TSTAR Tstar °Kelvin Effective Temperature of host star as reported in the KIC. If TSTAR_FLAG= 1, then TSTAR , LOGG, RSTAR derived using KIC J-K colour and linear interpolation of luminosity class V stellar properties of Schmidt-Kaler (1982).
TSTAR_FLAG Tstar Flag If Yes, then TSTAR, LOGG, RSTAR are derived using KIC J-K colour and linear interpolation of luminosity class V stellar properties of Schmidt-Kaler (1982).
LOGG Log(g) CGS units Surface gravity reported by KIC. If TSTAR_FLAG = 1, then log(g) is based on J-K interpolation. (cgs units)
KMAG Kepler Magnitude Kepler Magnitude Kepler Magnitude
CDPP CDPP Parts-per-million 6 hr Combined Differential Photometric Precision from Quarter 3
Status
VETTING Vetting Status Vetting status of KOI:
Confirmed: Confirmed and published planet.
Strong Candidate: Strong probability candidate, cleanly passes tests that were applied.
Moderate Candidate: Moderate probability candidate, not all tests cleanly passed but no definite test failures.
Insufficient Followup: Insufficient follow-up to perform full suite of vetting tests.
VETTING_FLAG Vetting Flag Numerical representation of Vetting Status
1: Confirmed and published planet.
2: Strong probability candidate, cleanly passes tests that were applied.
3: Moderate probability candidate, not all tests cleanly passed but no definite test failures.
4: Insufficient follow-up to perform full suite of vetting tests.
CONFIRMED Confirmed? If Yes, this KOI is a confirmed and published planet.
STRONG Strong Candidate? If Yes, this KOI is a strong probability candidate that cleanly passes tests that were applied.
MODERATE Moderate Candidate? If Yes, this KOI is a moderate probability candidate did not pass all tests cleanly but has no definite test failures.
NEEDS_FOLLOWUP Needs Followup? If Yes, there has been insufficient follow-up to perform a full suite of vetting tests.
SINGLE_TRANSIT Single Transit? If Yes, this KOI was detected on the basis of a single transit with the period derived from the transit duration and stellar radius.
EBPROB Eclipsing Binary Probability Probability of background eclipsing binary. Confused with planet's host star (see Borucki et al for discussion).
Observations
RECON Reconnaissance Spectra? If Yes, reconnaissance spectra of this KOI have been taken.
ADAPTIVE Adaptive Optics Observations? If Yes, adaptive optics observations of this KOI have been taken.
SPECKLE Speckle Observations? If Yes, speckle observations of this KOI have been taken.
RV10 10 m/s RV Spectra? If Yes, 10 m/s RV spectra of this KOI have been taken.
RV2 2 m/s RV Spectra? If Yes, 2 m/s RV spectra of this KOI have been taken.
NOOBS No Observations? If Yes, this KOI has not had any observations taken yet.
