Simple Histograms

In addition to setting logarithmic bins you can, also set the y-scaling to be logarithmic. This can be done by the Configure Axes section.

Advanced Histograms

Note that the statistics computed in the advanced mode tab do not update when the bin boundaries update. To compute statistics for a subset of the distribution, use a filter.

For additional documentation on the query language, including a list of available functions, a list of all available units, and a number of examples, visit the Filter Language help page.

For a list of all available columns in the Kepler database, visit the Data Columns help page.

Applying Filters

Note that multiple filters can be applied and displayed simultaneously to visualize various cuts on the histogram.
